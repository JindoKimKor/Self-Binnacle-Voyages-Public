## Why
- Starting point for understanding what frameworks exist for each language and which OS System Calls (APIs) they commonly use
- Gaining deeper insights into programming concepts through cross-language comparison (e.g., how different languages handle data types, memory, concurrency - and *why* they differ)
- Learning language-specific characteristics and best practices

![Language to OS/Platform API mapping](language-to-os-api-mapping.png)

## What
Comparative study and deep learning of programming languages (Python, C#, Java, JavaScript, etc.)

## How
- Cross-language comparisons → `resources/comparisons/`
- Language-specific deep dives → `resources/{language}/`
